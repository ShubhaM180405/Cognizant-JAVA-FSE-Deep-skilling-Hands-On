# Exercise 3 - Implementing Logging using Spring AOP

## Objective

To implement logging in a Spring application using Aspect-Oriented Programming (AOP) with Spring Framework.

---

- Spring Framework 5
- Spring AOP
- AspectJ Weaver
- Maven

## Description

This project demonstrates the implementation of Aspect-Oriented Programming (AOP) using Spring Framework. A logging aspect is created to intercept method execution before the business logic is executed.

The project uses Java-based Spring configuration and AspectJ annotations.

---

## Features

- Spring AOP
- Logging Aspect
- Annotation-Based Configuration
- Java Configuration
- Maven Project
