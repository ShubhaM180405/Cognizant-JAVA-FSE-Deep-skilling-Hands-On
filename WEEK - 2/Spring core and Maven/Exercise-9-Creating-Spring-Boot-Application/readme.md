# Exercise 9: Creating a Spring Boot Application

## Aim

To develop a Spring Boot-based Library Management application using Spring Web, Spring Data JPA, and H2 Database, and implement RESTful CRUD operations.

## Technologies Used

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- Maven
- IntelliJ IDEA
- Postman

## Project Structure

- controller
- entity
- repository
- service
- resources

## REST Endpoints

- POST /books
- GET /books
- GET /books/{id}
- PUT /books/{id}
- DELETE /books/{id}

## Output

- Spring Boot application started successfully.
- CRUD operations tested successfully using Postman.
- Data stored and retrieved using H2 in-memory database.

## Conclusion

Successfully created a Spring Boot application for a Library Management System and implemented RESTful CRUD operations using Spring Data JPA and H2 Database.
