# Exercise 4 - Creating and Configuring a Maven Project

## Objective

To create and configure a Maven project by adding the required Spring Framework dependencies and Maven Compiler Plugin.

---

## Technologies Used

- Java 21
- Maven
- Spring Context
- Spring AOP
- Spring Web MVC
- IntelliJ IDEA

---

## Project Structure

```
Exercise-4-Maven-Configuration
│
├── src

```

## Description

This project demonstrates Maven project configuration by adding Spring Framework dependencies and configuring the Maven Compiler Plugin. A simple Java program is included to verify that the project builds and runs successfully.

## Dependencies

- Spring AOP
- Spring Web MVC
