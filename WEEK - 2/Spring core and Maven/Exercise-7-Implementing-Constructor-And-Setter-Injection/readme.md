# Exercise 7 - Implementing Constructor and Setter Injection

## Aim

To implement Constructor Injection and Setter Injection in a Spring application for managing dependencies in the Library Management System.

---

## Scenario

The library management application requires both constructor and setter injection for better control over bean initialization and dependency management.

---

## Technologies Used

- Java 21
- Spring Framework 5.3.39
- Maven
- IntelliJ IDEA
