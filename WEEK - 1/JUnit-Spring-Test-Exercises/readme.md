# JUnit Spring Test Exercises

## Project Description
This project contains Spring Boot applications demonstrating JUnit 5 testing, Mockito, MockMvc, Integration Testing, Exception Handling, and Parameterized Testing.

## Technologies Used
- Java 21
- Spring Boot 3.5.16
- Maven
- Spring Web
- Spring Data JPA
- H2 Database
- JUnit 5
- Mockito
- MockMvc

## Exercises Completed
1. Unit Test for CalculatorService
2. Mockito Repository Test
3. MockMvc GET Endpoint Test
4. Integration Test with H2 Database
5. MockMvc POST Endpoint Test
6. Exception Handling Test
7. Custom Repository Query Test
8. Global Exception Handler Test
9. Parameterized Test

## How to Run
1. Clone the repository.
2. Open in IntelliJ IDEA.
3. Load Maven dependencies.
4. Run the JUnit test classes.
