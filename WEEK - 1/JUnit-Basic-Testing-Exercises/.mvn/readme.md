# JUnit Basic Testing Exercises

## Description
This project demonstrates the basics of unit testing using JUnit 4 and Maven.

## Exercises Completed

### Exercise 1
- Created a Maven project
- Added JUnit 4.13.2 dependency

### Exercise 2
- Created a Calculator class
- Wrote unit tests for:
    - Addition
    - Subtraction
    - Multiplication
    - Division

### Exercise 3
- Demonstrated different JUnit assertions:
    - assertEquals()
    - assertTrue()
    - assertFalse()
    - assertNull()
    - assertNotNull()

### Exercise 4
- Implemented the Arrange-Act-Assert (AAA) testing pattern
- Used @Before and @After annotations

## Technologies Used

- Java 21
- Maven
- JUnit 4.13.2
- IntelliJ IDEA
