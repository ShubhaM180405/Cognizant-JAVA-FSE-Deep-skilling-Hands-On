# Mockito Mock Dependencies Exercises

## Project Description

This project demonstrates Mockito-based testing in a Spring Boot application. It includes unit testing with mocked dependencies, controller testing using MockMvc, and integration testing using the H2 in-memory database.

## Technologies Used

- Java 21
- Spring Boot 3.5.16
- Maven
- Spring Web
- Spring Data JPA
- H2 Database
- Mockito
- JUnit 5
- MockMvc

## Exercises Completed

1. Mocking Repository Dependency using Mockito
2. Mocking Controller Dependency using MockMvc
3. Integration Testing using Spring Boot and H2 Database

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Download Maven dependencies.
4. Run the JUnit test classes.
