# SLF4J Logging Exercises

## Project Description

This project demonstrates the use of SLF4J (Simple Logging Facade for Java) with Logback for implementing logging in Java applications. It covers basic logging, parameterized logging, and configuring multiple appenders for console and file logging.

## Technologies Used

- Java 21
- Maven
- SLF4J
- Logback
- IntelliJ IDEA

## Exercises Completed

1. Logging Error Messages and Warning Levels
2. Parameterized Logging
3. Using Different Appenders (Console and File)

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Load Maven dependencies.
4. Run each Java class individually:
    - LoggingExample
    - ParameterizedLoggingExample
    - AppenderExample

## Output

- Console logging
- File logging (`app.log`)
