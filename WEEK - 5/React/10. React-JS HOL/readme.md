# 10. React-JS HOL

## Hands-on 10

### Objective

To demonstrate JSX syntax by rendering office space information, displaying images, iterating through objects, and applying conditional CSS styling.

---

## Software Requirements

- Node.js
- npm
- React
- Visual Studio Code

---


---

## Steps Performed

1. Created a React application.

```bash
npx create-react-app officespacerentalapp
```

2. Added an office image.

3. Created an array of office objects.

4. Displayed office details using JSX.

5. Used map() to render multiple office spaces.

6. Applied conditional CSS styling based on rent.

7. Executed the application.

```bash
npm start
```

---

## Output

The application displays office details including image, office name, rent, and address. Rent values are displayed in different colors based on their amount.
