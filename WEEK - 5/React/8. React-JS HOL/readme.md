# 8. React-JS HOL

## Hands-on 8

### Objective

To understand and implement React State using a class component by maintaining counters for people entering and leaving a shopping mall.

---

## Software Requirements

- Node.js
- npm
- React
- Visual Studio Code

---

## Project Structure

```
counterapp
│
├── public
│
├── src
│   ├── Components
│   │      └── CountPeople.js
│   │
│   ├── App.js
│   ├── App.css
│   ├── index.js
│   └── index.css
│
├── package.json
├── package-lock.json
└── node_modules
```

---

## Steps Performed

1. Created a React application.

```bash
npx create-react-app counterapp
```

2. Created the CountPeople component.

3. Initialized React State.

4. Implemented Login and Exit button functionality using setState().

5. Styled the application using CSS.

6. Executed the application.

```bash
npm start
```

---

## Output

The application displays the number of people entering and leaving the mall. Clicking the Login button increments the entry count, while clicking the Exit button increments the exit count.
