# 11. React-JS HOL

## Hands-on 11

### Objective

To understand React Event Handling by implementing button click events, multiple method invocation, passing arguments, synthetic events, and a simple Currency Convertor component.

---

## Software Requirements

- Node.js
- npm
- React
- Visual Studio Code

---

## Steps Performed

1. Created a React application.

```bash
npx create-react-app eventexamplesapp
```

2. Created the CurrencyConvertor component.

3. Implemented Increment and Decrement button events.

4. Demonstrated multiple method invocation using the Increment button.

5. Demonstrated passing arguments through the Say Welcome button.

6. Implemented a Synthetic Event using the Click on me button.

7. Created a Currency Convertor component using React event handling.

8. Executed the application.

```bash
npm start
```

---

## Output

The application displays:

- Increment and Decrement buttons
- Counter value
- Say Welcome button
- Click on me button
- Currency Convertor component with Amount and Currency fields
- Converted value after clicking Submit
