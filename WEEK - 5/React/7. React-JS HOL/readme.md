# 7. React-JS HOL

## Hands-on 7

### Objective

To demonstrate the use of React Props by passing shopping cart information from a parent component to a child component.

---

## Software Requirements

- Node.js
- npm
- React
- Visual Studio Code

---

## Project Structure

```
shoppingapp
│
├── public
│
├── src
│   ├── Components
│   │      ├── Cart.js
│   │      └── OnlineShopping.js
│   │
│   ├── App.js
│   ├── App.css
│   ├── index.js
│   └── index.css
│
├── package.json
├── package-lock.json
└── node_modules
```

---

## Steps Performed

1. Created a React application.

```bash
npx create-react-app shoppingapp
```

2. Created Cart and OnlineShopping components.

3. Passed shopping item data using Props.

4. Displayed all items in an HTML table.

5. Applied CSS styling.

6. Executed the application.

```bash
npm start
```

---

## Output

The application displays shopping items and prices in a bordered table with centered alignment.
